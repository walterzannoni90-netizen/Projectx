# Regole per agenti AI

- Non mostrare, registrare o committare secret, password, token o chiavi private.
- Non presentare saldi, depositi, indirizzi blockchain o rendimenti simulati come denaro reale.
- Le funzioni finanziarie reali richiedono un provider verificato, idempotenza, transazioni atomiche e audit log.
- Non abilitare `ENABLE_FINANCIAL_SIMULATION` in produzione.
- Ogni modifica deve superare le build di frontend e backend prima del merge.
